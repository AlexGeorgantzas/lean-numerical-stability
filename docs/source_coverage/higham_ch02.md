## Honest-strength notes

1. **Theorem 2.4 shape.** The primary source theorem now uses `fergusonMagnitudeExponentConditionLe`: representations of `x` and `y`, the printed bound `|x−y| < β^min(ex,ey)`, and a separate no-underflow proviso. The finite/normalized representation of `x−y` is proved internally by integer-lattice alignment. The older predicate that also carries a representation of `x−y` is retained only for compatibility with existing downstream proofs.
2. **Range hypotheses.** All (2.3)/(2.4)/(2.5)/(2.8-normal-branch) theorems for the concrete selector carry `finiteNormalRange` side conditions — these are the printed "lies in the range of F" / non-exceptional provisos, stated explicitly, not hidden strength loss. Underflow behavior is covered separately by the (2.8) branch theorems.
3. **Tie policy.** Theorem 2.2/2.3 are proved for *every* nearest-rounded value (relation-valued), so the results are independent of the tie rule, exactly as printed. The total selectors (`finiteRoundToEven`, `finiteNearestFl`) instantiate them.
4. **IEEE layer.** §2.3 coverage is a value/flag-level model (real-valued finite payloads), not a bit-level IEEE 754 encoding; hidden bit, formats table, and hardware history are SKIP-OK.
5. **Strict vs non-strict.** Model.lean's `FPModel` uses `|δ| ≤ u` — exactly printed (2.4). The concrete instance theorems prove the *stronger* strict `< u` on the normal range, consistent with Theorem 2.2's strict inequality.
