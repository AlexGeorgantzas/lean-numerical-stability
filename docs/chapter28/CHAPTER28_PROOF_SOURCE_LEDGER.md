# Chapter 28 Proof-Source Ledger

The chapter supplies citations rather than full proofs for several hard
properties. No citation is introduced as a Lean axiom. The Cauchy re-audit
removed transfers whose premises already contained the missing rational or
minor identity.  The Stewart re-audit likewise distinguishes an actual
Gaussian Householder producer from a lemma that only packages assumed
mass/support/invariance facts.  The Toeplitz/companion re-audit replaces
assumed trigonometric, coefficient, cyclicity, and low-rank identities with
the exact constructions currently proved and leaves the undisproved printed
endpoints OPEN below.  The Hilbert/Pascal re-audit removes assumption-only
asymptotic transfers, verifies the central-binomial Stirling endpoint, and
proves the actual all-orders Pascal perturbation and cube-root identities.
