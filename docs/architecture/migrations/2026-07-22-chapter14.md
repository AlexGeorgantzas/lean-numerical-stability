# Chapter 14 discrepancy migration

## Scope

This batch moves the Chapter 14 source-correction theorem out of the
transitional `NumStability.Higham` tree and into the canonical source tree.
The theorem name, namespace, signature, proof, and dependency on the reusable
matrix-inversion API are unchanged.

## Path map

| Historical module | Canonical module |
| --- | --- |
| `NumStability.Higham.Chapter14.Discrepancies` | `NumStability.Source.Higham.Chapter14.Discrepancies` |
| `NumStability.Algorithms.Ch14SourceCorrections` | `NumStability.Source.Higham.Chapter14.Discrepancies` |

Both historical modules are import-only compatibility wrappers governed by
[`../COMPATIBILITY.md`](../COMPATIBILITY.md).

## Boundary rationale

The declaration records a checked counterexample to the sign printed in
Higham Section 14.6. It is source correspondence, not a general numerical
algorithm. Its only mathematical dependency remains the reusable
`NumStability.Algorithms.MatrixInversion` module.

## Entry points and tests

`NumStability.Source.Higham.Chapter14` is the complete chapter aggregate.
One canonical-only smoke test imports the implementation leaf directly, and
two isolated old-only tests independently compile each historical path. The
canonical, source, source-Higham, and historical-Higham entry-point tests all
check the preserved theorem name.
