# Chapter 25 source migration

## Scope

This batch introduces the canonical `NumStability.Source` entry point and moves
the dependency-contained Chapter 25 nonlinear-system correspondence out of the
reusable-looking Algorithms tree. Declaration names and namespaces are
unchanged.

## Path map

| Historical module | Canonical module | Role |
| --- | --- | --- |
| `NumStability.Algorithms.Nonlinear.Higham25` | `NumStability.Source.Higham.Chapter25.NonlinearSystems` | source |
| `NumStability.Algorithms.Nonlinear.Higham25EigenClosure` | `NumStability.Source.Higham.Chapter25.Eigenproblem` | source |
| `NumStability.Algorithms.Nonlinear.Higham25Problem25_1` | `NumStability.Source.Higham.Chapter25.Problem01` | source |
| `NumStability.Higham` | `NumStability.Source.Higham` | compatibility entry point |

Every historical module is an import-only wrapper listed in
[`../COMPATIBILITY.md`](../COMPATIBILITY.md). Removal requires a declared
breaking release.

## Dependency rationale

The three modules contain only Higham Chapter 25 declarations. Before the move,
their only non-family production consumer was the Algorithms aggregate. The
canonical dependency chain is:

```text
Problem01 -> IterativeRefinement
NonlinearSystems -> Rounding, MatrixAlgebra, IterativeRefinement
Eigenproblem -> NonlinearSystems, DotProduct
```

No reusable module imports Chapter 25. At this chapter checkpoint,
`NumStability.Algorithms` retained one direct Chapter 25 aggregate import solely
to preserve its historical complete surface. The final graph has two reviewed
Algorithms-to-Source aggregate imports, for Chapters 24 and 25, and
`check_layout.py` ratchets that combined ceiling at two.

## Entry points and tests

The batch adds import-only `NumStability.Source`,
`NumStability.Source.Higham`, and
`NumStability.Source.Higham.Chapter25` aggregates. Canonical-only tests import
all three implementation modules directly. Three isolated old-only tests each
import exactly one historical wrapper. The historical Higham and Algorithms
entry-point tests check Chapter 25 declarations explicitly.
